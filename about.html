About SkySense

An AI-inspired weather UI that shows real-time weather, interactive maps and animated backgrounds.

Real-time Weather

SkySense fetches current weather using the OpenWeather API (temperature, humidity, wind, and condition).

Interactive Map

Search a city and see its location on a Leaflet map powered by OpenStreetMap tiles.

Animated UI

Animated weather card, dynamic backgrounds based on conditions, and smooth transitions for a clean demo experience.

Extensible

Easy to extend: add forecasts, geolocation, localStorage history, or EmailJS contact integration.

Technologies used

  • HTML5, CSS3, JavaScript
  • OpenWeather API (free tier)
  • Leaflet.js + OpenStreetMap
  • Animate.css for quick animations
  • Optional: EmailJS or GitHub Pages for hosting
